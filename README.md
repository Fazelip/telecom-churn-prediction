# Telecom Churn Prediction

## 🔍 Project Overview
Customer churn prediction using Logistic Regression, Random Forest, and XGBoost.  
This project analyzes customer churn in a telecom dataset and builds machine learning models to predict which customers are likely to leave.

## ⚙️ Models Used
- Logistic Regression  
- Random Forest  
- XGBoost  

## 📈 Key Results
- Random Forest achieved highest churn recall (~81%)  
- XGBoost provided balanced performance  
- Logistic Regression used as baseline  

## 🧠 Key Insights
- High monthly charges increase churn  
- Month-to-month contracts have highest churn rate  
- Customers without tech support are more likely to leave  

## 💡 Business Recommendations
- Apply price matching for high-risk customers  
- Offer bundles and loyalty programs  
- Target customers using churn prediction model
